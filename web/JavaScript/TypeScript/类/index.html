类

1. 类的定义：​

在 TypeScript 是不允许直接在 constructor 定义变量的 需要在 constructor 上面先声明

class Person {
    // 声明 变量
    name: string,
    age: number,
    construtor(name: string, age: number) {
        this.name = name;
        this.age = age;
    }
}

定义了变量不用 也会报错 通常是给个默认值 或者 进行赋值

2. 类的修饰符：​

总共有三个：

  1. public：任何地方 都可以访问（不写修饰符 也是 public）
  2. private ：只有 类内部 中可以访问
  3. protected：只有 类内部 和 子类 可以访问
class Person {
    // 声明 变量
    public name: string,
    private age: number,
    construtor(name: string, age: number) {
        this.name = name;
        this.age = age;
    }
}

4. interface 定义 类：​

ts interface 定义类 使用关键字 implements 后面跟 interface 的名字多个用逗号隔开 继承还是用 extends

interface PersonClass {
  get(type: boolean): boolean;
}

interface PersonClass2 {
  set(): void;
  asd: string;
}

class A {
  name: string;
  constructor() {
    this.name = "123";
  }
}

class Person extends A implements PersonClass, PersonClass2 {
  asd: string;
  constructor() {
    super();
    this.asd = "123";
  }
  get(type: boolean) {
    return type;
  }
  set() {}
}

5. 抽象类：​

应用场景如果你写的 类实例化之后毫无用处 此时我可以把他定义为抽象类

或者你也可以把他作为一个基类 -> 通过继承一个派生类去实现基类的一些方法

使用 abstract 声明一个类 是抽象类

abstract class A {
  name: string;
  constructor(name: string) {
    this.name = name;
  }
  print(): string {
    return this.name;
  }

  abstract getName(): string;
}
